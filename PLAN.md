# 现金流教育游戏 - 开发计划

## 游戏概述

基于《富爸爸穷爸爸》核心理念的回合制财商教育游戏。
玩家从"老鼠赛跑圈"开始，通过正确的资产/负债决策，让被动收入超过总支出，最终进入"快车道"实现财务自由。

**核心教学目标：** 让财商入门者直观理解——资产把钱放进口袋，负债把钱从口袋拿走。

## 技术方案

- **纯前端 SPA**：HTML5 + CSS3 + 原生 JavaScript（无框架依赖）
- **存档系统**：localStorage 实现多账号本地存档
- **部署方式**：静态文件，浏览器直接打开即可运行

## 核心玩法设计

### 1. 角色系统
玩家选择一个职业开局（如教师、程序员、医生），每个职业有不同的：
- 月薪收入（工资）
- 初始支出（房贷、车贷、生活费、信用卡等）
- 初始现金

### 2. 回合制流程（每回合 = 一个月）
```
发薪日 → 收入结算（工资 + 被动收入）→ 支出扣除 → 抽取事件卡 → 做出决策 → 更新财务报表
```

### 3. 事件卡系统
分为几大类：
- **投资机会卡**：房产、股票、基金、小生意（资产类）
- **额外支出卡**：意外支出、购买奢侈品诱惑（负债类）
- **市场波动卡**：资产涨价/跌价、有人高价收购你的资产
- **学习卡**：财务知识问答，答对获得奖励

### 4. 财务报表（核心 UI）
实时显示玩家的：
- **收入栏**：工资 + 各项被动收入（租金、分红、利息等）
- **支出栏**：房贷、车贷、信用卡还款、生活费等
- **资产栏**：房产、股票、基金、现金等
- **负债栏**：房贷余额、车贷余额、信用卡欠款等
- **月现金流** = 总收入 - 总支出（醒目显示正/负）

### 5. 胜利条件
**被动收入 ≥ 总支出** → 财务自由，逃出老鼠赛跑圈！

## 开发阶段

### 第一阶段：核心框架
1. 搭建 HTML 页面结构与 CSS 样式
2. 实现玩家数据模型（Player 类）
3. 实现职业选择系统
4. 实现财务报表 UI 渲染

### 第二阶段：游戏循环
5. 实现回合制游戏引擎（发薪、结算、事件）
6. 实现事件卡系统（数据 + 抽卡 + 决策）
7. 实现买入/卖出资产逻辑
8. 实现胜负判定

### 第三阶段：存档与体验
9. 实现 localStorage 多账号存档系统
10. 添加教学引导（首次游玩教程）
11. 添加财务知识提示（每张卡片附带教学说明）
12. UI 美化与动画效果

### 第四阶段：打磨
13. 平衡性调整（数值测试）
14. 响应式适配（手机/平板）
15. 整体测试与 Bug 修复
