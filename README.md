# 现金流 - 财务自由之路

基于《富爸爸穷爸爸》核心理念的回合制财商教育游戏。

## 游戏目标

让你的**被动收入**超过**总支出**，实现财务自由，逃出老鼠赛跑圈！

## 核心概念

- **资产**：把钱放进你口袋的东西（出租房产、股票分红、生意利润）
- **负债**：把钱从你口袋拿走的东西（房贷、车贷、信用卡）
- **财务自由** = 被动收入 ≥ 总支出

## 如何运行

直接用浏览器打开 `index.html` 即可开始游戏，无需安装任何依赖。

## 玩法

1. 选择职业开局（不同难度）
2. 每回合代表一个月：领工资 → 收被动收入 → 扣支出 → 抽事件卡 → 做决策
3. 通过投资积累资产，增加被动收入
4. 避免不必要的消费和负债
5. 当被动收入 ≥ 总支出时，恭喜你实现了财务自由！
