游戏说明

目标

在规定时间内让你的被动收入超过总支出，实现财务自由！可选60/120/150/180个月。

什么是资产？

资产是把钱放进你口袋的东西：出租房产、股票分红、生意利润。

什么是负债？

负债是把钱从你口袋拿走的东西：房贷月供、车贷月供、信用卡还款。

核心公式

月现金流 = 总收入（工资 + 被动收入） - 总支出

游戏流程

每回合 = 一个月：领工资 → 收被动收入 → 扣除支出 → 抽取事件卡 → 做出决策。每年会有通货膨胀，生活费自动上涨！

进阶策略

你可以随时卖出资产或提前还贷。投资机会会随时间解锁更高级的选项。注意分散投资和现金储备！

多卡选择

每月你可以从2-3个事件中选择一个处理。20月后解锁3选1。社交资本高时，投资机会更丰富！

主动行动

每月可以执行一次主动行动：搜索投资、自学课程、参加社交聚会。不再被动等待机会！

社交资本

社交资本影响你的投资渠道。拒绝社交会降低社交资本，投资机会也会减少。适度社交是一种"软性投资"。

职业特性

每种职业有独特优势：服务员节俭（消费减半）、教师学者（学习奖励翻倍）、程序员技术（科技投资+20%）、医生人脉（社交带来投资机会）。

资产互动

持有资产后会触发互动事件：租客谈判、批量装修、商业合作等。你的资产越多，互动越丰富，这就是规模化管理的力量。

投资预览

每次投资决策前，系统会显示买入后的现金流预览和剩余资金，帮你做出明智选择。心态低落时信息会模糊，崩溃时投资成本+20%！

象限进化

游戏中会显示象限进化条件。E→S需要现金+生意，S→B需要3个生意+被动收入≥5000，B→I需要被动收入≥支出80%。满意度低于30无法进化！

贷款系统

你可以通过主动行动向银行申请贷款。贷款额度和利率取决于你的芝麻信用（350-950分）。按时还贷提升信用分，逾期则大幅下降。用贷款买资产是"好负债"，用来消费是"坏负债"。

多重行动

每月你有2次主动行动机会，可从以下7种行动中选择：

🔍 搜索投资 - 花时间跑中介看盘，寻找投资标的
📖 自修课程 - 花¥1,200报名理财课程，60%获得答题机会
🤝 社交聚会 - 花¥300请客吃饭，恢复满意度和社交资本
🏦 申请贷款 - 根据信用评分获取资金
💪 兼职打工 - 下班后赚外快，收入因职业而异
🧘 休息调整 - 恢复满意度（低满意度时效果更好）
📊 市场调研 - 花¥800买行业报告，下月事件卡多一个选择

芝麻信用

芝麻信用范围350-950分，初始650分。评分越高，贷款利率越低（950分=4%，650分=8%）。按时还贷+10~15分，逾期-50分。善用信用杠杆购买正现金流资产是加速财务自由的关键。

小贴士

记住富爸爸的教诲：富人买入资产，穷人和中产买入自以为是资产的负债。关注净现金流而非总价，利用好负债撬动资产增长。
